# Deployment Checklist ✅

Use this checklist to ensure everything is ready for GitHub Pages:

## Pre-Deployment

- [x] All HTML files use relative paths (no absolute paths)
- [x] All CSS files are linked correctly
- [x] All JavaScript files are linked correctly
- [x] Photos are in the `photos/` folder
- [x] Photo paths in `photos.js` match actual file names
- [x] `.gitignore` file is created
- [x] `README.md` is updated for GitHub

## Files to Upload

Make sure these files are in your repository:

### Core Files
- [ ] `index.html` - Main homepage
- [ ] `photos.html` - Photo gallery page
- [ ] `styles.css` - Main stylesheet
- [ ] `photos.css` - Photo gallery styles
- [ ] `script.js` - Main page JavaScript
- [ ] `photos.js` - Photo gallery JavaScript
- [ ] `README.md` - Project documentation
- [ ] `.gitignore` - Git ignore rules
- [ ] `GITHUB_SETUP.md` - Setup instructions

### Assets
- [ ] `photos/photo1.jpg`
- [ ] `photos/photo2.jpg`
- [ ] `photos/photo3.jpg`
- [ ] `photos/photo4.jpg`
- [ ] `photos/photo5.jpg`
- [ ] `photos/photo6.jpg`
- [ ] `photos/photo7.jpg`
- [ ] `photos/photo8.jpg`
- [ ] `photos/photo9.jpg`
- [ ] `photos/photo10.jpg`
- [ ] `photos/photo11.jpg`

## GitHub Pages Setup

- [ ] Repository created on GitHub
- [ ] All files uploaded to repository
- [ ] GitHub Pages enabled in Settings
- [ ] Branch set to `main` (or `master`)
- [ ] Root folder selected
- [ ] Website URL noted: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Testing

- [ ] Homepage loads correctly
- [ ] Photo gallery page loads
- [ ] All photos display correctly
- [ ] Navigation works (home ↔ gallery)
- [ ] Lightbox works when clicking photos
- [ ] Responsive design works on mobile
- [ ] All animations work
- [ ] No console errors (check F12)

## Post-Deployment

- [ ] Share the GitHub Pages URL
- [ ] Test on different browsers
- [ ] Test on mobile devices
- [ ] Update README with actual GitHub Pages URL

**Ready to deploy?** Follow the instructions in `GITHUB_SETUP.md`!
